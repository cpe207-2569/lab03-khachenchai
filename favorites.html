วง Clash

ผมชอบฟังเพลงวง Clash มาก ๆ ฟังทุกอัลบั้มร้องได้แทบทุกเพลง [เคยบอกกับครอบครัวไปว่า ไม่ว่าจะอายุเท่าไร แต่ขอให้ในงานขาว-ดำของตนเองเปิดเพลงวง Clash ด้วย]

Tailwind CSS

Tailwind CSS เป็น CSS Utility ที่ผมชอบใช้มาก ๆ เนื่องจากเป็นเครื่องมือที่ชื่อ class เข้าใจง่าย และตรงตัวกว่า Bootstrap [ในความคิดส่วนตัว] รวมทั้งมันสามารถ Custom ตามใจชอบได้ง่ายมาก ๆ

Fullstack Dev

กระผมมีความสนใจในด้านการพัฒนา Applictions เป็นหลัก รวมทั้งสามารถเขียนเว็บแอพ โดยใช้ HTML CSS JS React(NextJS) รวมทั้ง Backend ด้วย express.js [แต่จะถนัดไปทาง Frontend มากกว่า]

Roblox

Roblox เป็นเกมเดียวที่ผมเล่นเป็นหลัก เนื่องจากมีความหลากหลายในแนวเกม ซึ่งทำให้ไม่จำเป็นต้องไปหาเกมอื่น ๆ ที่ไหนเล่นอีก และตอน ป.5 ผมก็ได้เริ่มฝึกเขียนโปรแกรมเป็นครั้งแรกก็ด้วย Roblox Studio นี่แหละ
